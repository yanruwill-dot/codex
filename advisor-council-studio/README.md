# Advisor Council Studio

一个可直接打开的静态网页，把一个真实商业问题拆成 5 位顾问视角，并生成可执行建议与渠道文案。

## What it does

- 输入一个业务问题。
- 选择营销增长、产品设计、市场预测、公关破局或战略选择。
- 用 5 个公开商业人物的思维镜头拆解问题：
  - Sam Altman: consensus and narrative
  - Jeff Bezos: customer-backed growth
  - Steve Jobs: product focus
  - Jensen Huang: market prediction
  - Elon Musk: public proof and breakthrough
- 输出具体建议、不要做的事、验证指标、关键追问。
- 生成可直接使用的朋友圈文案、口播文案、短视频文案、社群文案。

## Run locally

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8732
```

Then visit:

```text
http://127.0.0.1:8732/
```
