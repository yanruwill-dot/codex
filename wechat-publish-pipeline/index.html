发布链路

1. 生成文章

2. Image API 配图

用页面生成的 prompt 去 Image API / Image2 生成封面和正文图。

3. 公众号排版

复制 HTML 到公众号编辑器或发布脚本，发布前清理内部标签。

4. 保存/发表/核验

明确区分草稿、已发表和公开页核验，不能只停在“生成好了”。
