一键链路

选题判断

输入行业、受众、目标和素材，先判断为什么值得写、读者焦虑是什么、标题怎么制造点击理由。

Image API 配图

封面图和正文图必须走 Image API / Image2，保留 prompt、生成结果和失败边界。

排版清理

删除“作者、封面、draft、placeholder”等内部词，正文只保留读者需要看的内容。

保存草稿

优先公众号官方 API；不可用时再走后台 UI，并截图保存草稿状态。

正式发布

只有用户明确说发表/发出去，才进入发布；遇到扫码、手机确认、实名拦截就停下说明。

公开核验

发布后必须拿公开页或后台状态回查，不能只说“应该发了”。

接口边界

图片 API Key 不写进公开页面。实际执行时在本地或后端读取环境变量，前端入口只作为工作流说明和跳转中控。
