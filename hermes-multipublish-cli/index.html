CLI 作战顺序

账号检查

先读账号状态：登录、实名、设备、代理、平台权限，不能把账号阻断当内容失败。

内容打包

每个平台拆出标题、正文、图片、视频、话题、封面和发布时间。

发布回执

记录 publish_status、平台原始返回、公开 URL 或需要人工处理的阻断。

本地约定

优先读取本机 Hermes 环境配置，不在公开页面展示密钥。发布完成态要区分：本地包、后台草稿、已发表、公开页已核验。
