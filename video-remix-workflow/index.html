VIDEO REMIX / SAFE REWRITE

一键修改别人视频：拆结构，不搬运

这个入口做的是授权素材或对标视频的结构化改写：提取选题、开头钩子、镜头节奏、口播逻辑和转化动作，再生成新的脚本、分镜、配音和画面建议，不复制原视频素材。

先抓对标复盘 生成全平台脚本
拆提取节奏、镜头、钩子、卖点和 CTA。
改换行业、换案例、换人设、换表达。
生生成新脚本、新分镜、新封面提示词。
发交给 Hermes / 蚁小二多平台发布。

一键改写步骤

上传或粘贴链接

优先使用自己素材、授权素材、公开可分析链接；没有授权时只做结构学习。

抽取脚本骨架

拆 3 秒钩子、问题放大、证明材料、转折、行动号召和评论区引导。

生成新版本

换成自己的产品、案例和人物口吻，输出口播、镜头清单、字幕和封面文案。

合规检查

检查是否复用原视频画面、音乐、人脸、商标和独创表达，风险高就改成全新拍摄方案。

生产素材

用 Image API 生成封面和贴片，用 TTS/剪辑工具生成可发布素材。

多平台发布

通过 Hermes / 蚁小二记录发布结果和平台阻断。

边界

不提供搬运、去水印、规避版权检测或冒充原作者的流程。可做的是“对标学习 + 新内容生产”。
