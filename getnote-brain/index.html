管理入口

知识库要有边界

笔记 ID 必须来自 API 返回；异步任务不能跳过轮询；内链和分享链接不能混用。素材属于用户隐私，不能随便公开展示。

给 Codex 的调用提示

这不是公开 API 密钥说明，只是以后调用 GetNote skill 时的任务边界。

用「得到大脑（Get笔记）」处理： 动作：保存 / 搜索 / 查看 / 加标签 / 知识库管理 内容：{{链接、图片、文字或关键词}} 要求： - 先检查 GETNOTE_API_KEY 和 GETNOTE_CLIENT_ID 是否存在 - Base URL 使用 https://openapi.biji.com - note_id 按字符串处理，不编造 ID - 链接/图片保存拿到 task_id 后必须轮询任务进度 - API 返回 success:false 时报告真实错误，不假装已保存
长期判断：内容工厂真正值钱的不是一次生成，而是把素材、判断、案例和风格沉淀成可召回的资产。得到大脑负责让这些东西留下来。
