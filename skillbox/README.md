# 颜汝的 AI 技能箱落地页

本地静态页面，视觉参考 Sensoria 的黑底、霓虹、AI Beauty / Skin Analysis 气质，主视觉和漂浮卡片已替换为用户指定的颜汝照片裁切版本。

打开方式：直接打开 `index.html`，或在本目录启动本地静态服务。

主要内容：

- 首屏：品牌名、汉堡菜单、年份刻度、人物主视觉、漂浮图片卡片、底部超大品牌标题。
- 技能矩阵：content-system、images2 / imagegen、product-design、data-analytics、playwright / browser、mcp-tools。
- 全部 Skills 表格：`skills-data.js`，共 282 条，展示名字和中文功能描述，并支持搜索。
- MCP 层：把外部工具连接成「真实世界仪器」。
